# Chat Application

## 📌 Description
This is a real-time chat application developed using Java Socket Programming. It enables communication between multiple users over a network.

## 🚀 Features
- Real-time messaging
- Client-server architecture
- Multi-user communication
- GUI using AWT

## 🛠 Technologies Used
- Java
- AWT
- Socket Programming (TCP)

## ▶️ How to Run
1. Open project in IDE
2. Run server file
3. Run multiple client files
4. Start chatting between users
